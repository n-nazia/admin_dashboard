# Admin Dashboard Web Application

A responsive **Admin Dashboard frontend project** built using **HTML, CSS, and Vanilla JavaScript**.  
This project demonstrates core frontend concepts such as user management, localStorage data persistence, search functionality, and interactive UI components.

##  Features

-  Login system (client-side validation)
-  Add new users
-  Edit user details
-  Delete users
-  Search users instantly
-  Active / Inactive user status
-  Reports section with:
  - Total users count
  - Summary cards
  - Simple CSS-based charts
-  Persistent data storage using **localStorage**
-  Responsive UI design

##  Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling, layout, and responsive design  
- **JavaScript (ES6)** – Application logic and DOM manipulation  
- **Browser localStorage** – Permanent data storage (no backend)
